대학생 관리 프로그램 요구사항 작성

대학생
- 대학생은 학번, 이름, 전공, 학년, 연락처를 가지고 있다
- 학번은 년도 4자리 전공 코드 3자리 이름 순서 3자리를 이용하여 학번을 구성한다
- 전공은 복수 전공을 허용하지 않고, 문자열로 구성한다
- 연락처는 전화번호 1개, 주소, 상세주소로 구성된다.


강의
- 강의는 강의 번호, 강의명, 담당교수, 강의실, 강의시간, 학점, 시수, 전공여부를 가지고 있다.
- 강의번호는 정수로 순차적으로 증가하도록 구성한다
- 강의명, 담당교수는 문자열로 구성한다
- 강의실은 건물명 강의실명으로 된 문자열 순으로 구성한다
- 강의시간은 1A, 1B와 같은 형태로 구성한다
    - A는 정각, B는 30분, 1A는 9:00를 의미한다
    - 월 1A, 1B, 화 3A,3B는 월요일 9시에서 10시까지, 화요일 11시에서 12시까지 수업을 의미한다.
- 학점은 강의를 이수할 때 획득할 수 있는 학점으로 정수로 구성한다
- 시수는 1주일에 진행하는 강의 시간을 의미하며 정수로 구성한다
- 전공 여부는 전공 선택, 전공 필수, 교양 선택, 교양 필수를 값으로 가지며 문자열로 구성한다


교수
- 교수는 교수번호, 교수명, 교수실, 전공을 가지고 있다
- 교수번호는 년도 4자리, 과 코드 3자리, 순서 3자리로 구성한다
- 교수명은 문자열로 구성한다
- 교수실은 건물명 교수실 호수로 된 문자열로 구성한다
- 전공은 문자열로 구성하고, 복수 전공은 불가능하다




기능적인 측면

수강 신청
- 대학생은 강의를 수강 신청 할 수 있다
- 수강 신청 시 강의 번호를 선택해서 수강 신청할 수 있다.
- 수강 신청 시 동 시간대에 강의를 신청할 수 없다
- 이미 수강 신청한 강의를 다시 신청할 수 없다

수강 취소
- 대학생은 수강 신청한 강의를 수강 취소할 수 있다
- 수강 취소 시 수강 신청한 강의에서 강의 번호를 선택해 수강 취소를 할 수 있다
- 수강 신청하지 않은 강의는 수강 취소를 할 수 없다

강의 등록
- 관리자는 강의 정보를 입력해서 강의를 등록할 수 있다
- 강의 번호가 중복된 강의는 등록할 수 없다
- 강의 등록 시 같은 시간, 같은 강의실에서 진행하는 강의는 등록할 수 없다
- 강의 번호 이외에 정보는 중복될 수 있다
    - 같은 과목인데 담당 교수가 다른 경우
    - 같은 교수가 여러 강의를 하는 경우
    - 같은 강의실에서 여러 강의가 진행
    - 같은 시간대에 여러 강의가 진행

강의 수정
- 관리자는 등록된 강의를 수정할 수 있다
- 강의 수정 시 강의 번호를 제외한 모든 정보를 수정할 수 있다
- 강의 수정 시 같은 시간, 같은 강의실에서 진행하는 강의로 수정할 수 없다.

강의 삭제
- 관리자는 등록된 강의를 삭제할 수 있다.



학생/교수 관리

학생/교수 등록
- 관리자는 학생/교수 정보중 (모든 학생 정보)/(교수실을 제외한 교수 정보)를 입력해서 등록할 수 있다
- 관리자는 학생/교수 번호를 지정된 규칙에 맞게 부여한다

학생/교수 수정
- 학생/교수는 학생/교수 정보를 수정할 수 있다
- 학생번호/교수번호를 제외한 정보를 수정할 수 있다.

학생/교수 삭제
- 없음

쇼핑몰 사이트 요구사항 명세서

분류 관리
-분류 등록
    - 관리자는 제품의 분류를 등록할 수 있다
-분류 수정
-분류 삭제

제품 등록
- 관리자는 제품 정보를 입력해서 제품을 등록할 수 있다
- 제품 정보에는 제품 코드, 제품명, 제품 상세내용, 제품 가격과 옵션, 제품 썸네일로 구성
- 옵션에는 제품 사이즈가 들어가는데 제품 종류에 따라 제품 사이즈 종류가 달라짐
    - 제품이 옷인 경우 XS, S, M, L, XL로 구성
    - 제품이 신발인 경우 220~300까지 5단위로 구성
- 분류는 대분류만 적용하고 대분류는 옷, 신발, 모자, 악세서리, 기타로 구분
- 제품 썸네일은 이미지이며, 최대 5개 선택이 가능하고, 최소 1개를 선택해야 한다


제품 수정
- 관리자는 제품 정보를 수정할 수 있다
- 관리자는 제품 옵션을 수정할 수 있다
    - 옵션 수정 시 수정할 제품을 선택하고, 수정할 옵션을 선택해서 수정한다
- 수정 가능한 제품 정보는 모든 제품정보이다.

제품 삭제
- 관리자는 등록된 제품 정보를 삭제할 수 있따
- 제품 삭제 시 제품과 관련된 옵션들, 썸네일들도 같이 삭제가 된다

제품 조회
- 사용자는 분류를 이용해서 원하는 분류에 맞는 제품들을 조회할 수 있다
- 사용자는 검색을 통해 검색어가 포함된 제품들을 조회할 수 있다

제품 상세 조회
- 사용자는 제품 조회에서 조회한 제품들을 상세 조회할 수 있다
- 상세 조회 시 제품 썸네일들이 보여지며, 제품 정보들을 조회할 수 있다
- 

장바구니
제품 추가
- 회원은 제품을 상세 조회해서 원하는 제품을 장바구니에 담을 수 있다
    - 제품의 옵션과 수량을 선택해서 장바구니에 담을 수 있다
    - 여러 종류의 옵션을 한번에 장바구니에 담을 수 없다

제품 삭제
- 회원은 장바구니에 담긴 제품을 삭제할 수 있다

제품 수정
- 회원은 장바구니에 담긴 제품의 수량을 수정할 수 있다


제품 구매
 바로 구매
- 회원은 제품 상세 조회에서 제품의 옵션과 수량을 선택해서 바로 구매를 통해 한 제품을 구매할 수 있다
 장바구니 구매
- 회원은 장바구니에 담은 제품들 중에서 선택해서 여러 제품을 한번에 구매할 수 있다
- 여러 제품을 한번에 구매할 때 각 제품마다 구매 번호가 할당이 된다


구매 취소
- 회원은 구매 제품 내역에서 배송 전인 제품에 대해 구매를 취소할 수 있따
- 취소 시 결제한 금액을 환불한다
- 취소 시 구매 취소한 제품을 구매 목록에서 제거한다
- 취소 시 구매 취소한 제품은 구매 취소 목록에 추가한다

제품 환불
- 회원은 배송 된 제품에 문제가 있을 시 제품을 환불 받을 수 있다
- 제품 환불 시 환불 한 제품에 판매자에게 배송이 되고 제품 불량이 확인이 되면 관리자는 해당 제품을 환불 한다
- 환불한 제품은 구매 목록에서 제거 되고, 환불 목록에 추가된다

제품 교환
- 회원은 배송 된 제품에 문제가 있을 시 제품을 교환 받을 수 있다
- 제품 교환 시 교환 한 제품이 판매자에게 배송 되고, 제품 불량이 확인 되면 관리자는 제품을 교환해 준다


구매 확정
- 회원은 배송 된 제품에 문제가 없는 경우 구매 확정을 할 수 있다

회원관리
- 회원가입
    - 사용자는 아이디, 비번, 이메일, 전화번호를 입력하여 회원가입을 할 수 있다
    - 아이디는 영문자, 숫자만 가능하며 8~13자 이다
    - 아이디는 중복 검사를 통해 중복되지 않은 경우에만 사용할 수 있다.
    - 비밀번호는 영문자, 숫자, 특수문자(!@#$%)만 가능하며, 8~20자이다
    - 이메일은  이메일 형식에 맞게 작성한다
    - 전화번호는 전화번호 형식에 맞게 작성한다
    - 아이디, 비번, 이메일, 전화번호는 필수 항목이다

- 로그인
    - 사용자는 아이디, 비번을 입력하여 로그인 할 수 있다
    - 아이디, 비밀번호가 일치하면 로그인이 된다
    - 아이디, 비밀번호가 일치하지 않으면 알림 메세지를 띄운다
    - 로그인 5회 실패 시 비밀번호 찾기를 통해 비밀번호를 재 발급 받아야 로그인이 가능하다
    - 로그인 성공 시 실패 횟수가 0으로 초기화 된다.

- 로그아웃
    - 회원은 로그아웃 할 수 있다

- 비밀번호 찾기
    - 회원은 아이디와 이메일을 입력하여 비밀번호를 찾을 수 있다
    - 입력한 아이디와 이메일이 회원 가입 시 입력한 아이디, 이메일과 일치하면 메일로
    비밀번호를 변경할 수 있는 페이지의 링크를 전송한다.
    - 일치하지 않으면 알림 메세지를 띄운다
    - 이메일에서 보낸 링크를 클릭하면 새 비밀번호를 변경할 수 있는 페이지로 이동한다
    - 이동한 페이지에서 새 비밀번호와 새 비밀번호 확인을 입력하면 비밀번호가 변경된다

- 마이 페이지
    - 회원은 마이 페이지에서 회원 정보를 확인할 수 있다
    - 마이 페이지에서는 구매 목록, 구매 취소 목록, 반품 목록, 환불 목록, 구매 완료 목록을 조회할 수 있다
    - 회원 정보를 수정할 수 있다
        - 회원 정보는 비밀번호, 이메일, 전화번호를 수정할 수 있다.
